A LANDSLIDE HAS OCCURRED

In the 1999 Real Time Strategy PC video game LEGO® Rock Raiders, whenever a landslide occurs Chief exclaims "A landslide has occurred!" as a warning message to the player. On modern computers, these two events have create an annoying glitch where Chief's audio clip will repeat in an endless loop until the game is closed or crashes by way of another glitch.

A LANDSLIDE HAS OCCURRED recreates this repetitive and often parodied glitch in addition to paying homage to the seemingly endless amount of sandwiches Rock Raiders constantly consume.

CRTC

CRTC is my first experiment with JavaScript. Written using pure JavaScript, it calculates approximate render times when rendering a graphic using the Blender Cycles Engine with GPU acceleration enabled.

jQuery Spoiler

Developed from personal usage and subsequently released, jQuery Spoiler is a minimal and lightweight jQuery plugin that hides content from public view until th toggle button is clicked, often referred to as a spoiler. Requiring only jQuery >= 1.10.0 and a single CSS property, jQuery Spoiler is easy to setup and leaves all design aspects up to you.

TEE VEE HERE

When the 2000 Strategy PC video game LEGO® Alpha Team is run on Windows XP computers,a very well documented and sometimes so annoying it is funny audio glitch occurs where one of Tee Vee's audio clips will become stuck in an endless loop, crippling all game play unless both music and sound effects are disabled.

As my 2014 April Fool's joke, I adapted my A LANDSLIDE HAS OCCURRED project into TEE VEE HERE, recreating Tee Vee's everlasting annoyingness in a play-it-everywhere website.

Tic-tac-toe

In 2014 September, my college advisor and I were talking about possible JavaScript projects for class. I had come across a Tic-tac-toe game written by Graham Lavender a few days before, so I showed it to him. While playing with it, we noticed many issues that are present in an actual Tic-tac-toe game. My advisor then asked me to make a Tic-tac-toe gamer using pure JavaScript that addressed these shortcomings, and I did just that.

You can get more details on the game and my experience writing the game on on my blog.
