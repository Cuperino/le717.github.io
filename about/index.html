About Triangle Land

Triangle Land is a Jekyll-powered, open-source HTML5 website hosted on GitHub Pages. I have written everything by hand, with much of the design by rioforce. The intent of this site is to serve as a live playground for items I cannot host on WordPress.com and to help me better learn client-side web development.
