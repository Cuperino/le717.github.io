Code Snippet

var $imgSrc = $(".ie-svg").attr("src");
    $(".ie-svg").attr("src", $imgSrc.substring(0, $imgSrc.length - 3) + "png");

This small bit of jQuery simply gets the src attribute of the ie-svg class
and replaces it with a PNG image counterpart already on the server.
It works well for working around some of Internet Explorer's issues with SVG vector graphics.
